==Lundakarnevalen 2014 (Futuralkarnevalen)

This is the internal website and API for {Lundakarnevalen}[http://www.lundakarnevalen.se/] 2014, a large student carnival arranged every fourth year.

The Ruby on Rails project includes a wide variety of functionalities, for example:
 
* Different tools for registration and administration of "karnevalister" (students that take part in the carnival)
* Order management system
* API for communication between iOS/Android apps and the internal website
* Tool for sending push notifications to Android and iOS app users
